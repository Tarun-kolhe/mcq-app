# MCQ Application

A full-stack MCQ (Multiple Choice Questions) application that allows admins to manage quiz questions and users to take tests with real-time accuracy results. The application is deployed live and also converted into a mobile Android app using PWA technology.

##Features

### Admin Module
- Secure admin login
- Add MCQ questions with multiple options
- Mark correct answers
- Delete existing questions
- All changes reflect instantly in the test module

### User Module
- Start test with latest available questions
- Answer MCQs in a simple, clean UI
- View results after test completion
- Accuracy percentage calculated and displayed

### General
- Responsive design (Desktop & Mobile)
- Persistent data storage
- Installable as a Progressive Web App (PWA)
- Available as an Android APK

##  How It Works

1. Admin logs in and manages MCQ questions
2. Questions are stored in the database via backend APIs
3. Users start the test and answer questions
4. System evaluates responses
5. Accuracy and results are displayed after completion
6. Same web app is wrapped and delivered as a mobile app
